# Lab-04: Segurança Dinâmica (DAST)

Este repositório faz parte do workshop de DevSecOps, focado em análise de segurança em tempo de execução.

## Objetivo
Demonstrar a aplicação de testes dinâmicos (DAST) utilizando o **OWASP ZAP** integrado ao GitHub Actions para identificar vulnerabilidades em aplicações rodando.

## Tecnologias
- **GitHub Actions**: Automação de Pipeline.
- **Node.js (Express)**: Aplicação alvo.
- **OWASP ZAP**: Scanner de vulnerabilidades dinâmicas.

## Segurança
Este projeto contém vulnerabilidades propositais para fins educacionais. 
- **SAST**: Análise de código estático (ver Labs anteriores).
- **DAST**: Análise de comportamento em execução (neste Lab).
